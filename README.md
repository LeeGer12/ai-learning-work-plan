# DailyFlow — 工作日报助手

一个轻量的每日工作管理工具，帮你快速记录待办、生成日报。零依赖，双击即用。

## 功能一览

| 功能 | 说明 |
|------|------|
| 📋 待办管理 | 添加、完成、删除待办，支持备注 |
| 📅 日历导航 | 左侧可视化日历，点击日期切换，有记录的日期带表情标记 |
| 📄 生成日报 | 一键生成日报文本，可编辑后复制到剪贴板 |
| 🏷️ 项目标签 | 待办按项目归类（支持自定义项目） |
| 🔴 优先级 | 紧急/重要/普通三级，自动排序 |
| 🔍 搜索筛选 | 跨日期搜索，关键词高亮 |
| 📊 周统计 | 本周完成率柱状图、完成数、项目数 |
| ⚡ 快捷模板 | 预设常用工作模板，一键添加 |
| 🕐 时间轴 | 记录当天操作历史，可删除单条 |
| ↕️ 拖拽排序 | 拖拽调整待办顺序，或一键按优先级排序 |
| 💾 数据备份 | 导出/导入 JSON 备份文件，每日提醒备份 |
| ⏭️ 待办顺延 | 将当前日期未完成待办手动复制到明日，带来源标记且防重复 |

## 使用方法

1. 下载 [`code/dailyflow/dailyflow.html`](code/dailyflow/dailyflow.html)
2. 双击打开（Chrome 或 Edge）
3. 开始使用

不需要安装任何东西，不需要联网。

## 数据存储

所有数据存在浏览器 `localStorage` 中。关闭浏览器不会丢失数据，但清除浏览器缓存会丢失。

**建议**：定期点击「💾 导出备份」保存数据。应用会在超过 1 天未备份时自动提醒。

## 技术方案

- 原生 HTML + CSS + JavaScript，单文件实现
- 零外部依赖
- 浏览器 localStorage 存储
